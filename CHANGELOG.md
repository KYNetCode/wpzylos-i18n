### Added

- Added `I18n::setScriptTranslations()` for WordPress JavaScript translation JSON files.
- Added `I18n::setScriptTranslationsFor()` for registering multiple script handles.
- Added framework-neutral JavaScript i18n helpers backed by WordPress `wp.i18n`.
- Added Vue plugin support for template-level `__('Original text')` usage.
- Added React provider, hook, and HOC support for WordPress gettext strings.
